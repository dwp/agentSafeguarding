Add any comments or recommendations that you think will help decision makers set appropriate safety measures.

For example, a decision maker might decide:

  • to send a warning letter to this customer
  • that this customer should only have appointments in a screened environment
  • that staff should only see this customer when accompanied by colleagues
  • that this customer should not have any home visits
  • that staff should only deal with this customer by post
  • to ban this customer from a particular location

These are only examples - a decision maker can take actions that are not in this list.

A decision maker will review your comments and recommendations before they make a decision.
