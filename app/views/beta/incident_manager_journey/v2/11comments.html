Add comments and recommendations

Add any comments or recommendations that you think will help decision makers set appropriate safety measures.

For example, a decision maker might recommend:

  • sending a warning letter to this customer
  • only arranging appointments with this customer in a screened environment
  • that staff should only see this customer when accompanied by colleagues
  • not arranging any home visits with this customer
  • only dealing with this customer by post
  • banning this customer from a particular location

These are examples only - a decision maker can make recommendations that are not in this list.

A decision maker will review your comments and recommendations before they make a decision about this incident.
